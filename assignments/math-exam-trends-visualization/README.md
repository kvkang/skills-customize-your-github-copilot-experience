# 📘 Assignment: Math Exam Trends Visualization

## 🎯 Objective

In this assignment, you will analyze math scores from multiple exams and create visualizations that show how student performance changes over time. You will practice working with CSV data, summary statistics, and trend charts in Python.

## 📝 Tasks

### 🛠️	Analyze Math Exam Data

#### Description
Load the provided CSV file containing student math scores across multiple exams. Explore the dataset and calculate summary information that helps you understand student and class performance over time.

#### Requirements
Completed program should:

- Load the CSV file using Python (for example, with pandas).
- Display the first 5 rows of the dataset.
- Calculate summary statistics for each exam score column.
- Calculate the class average for each exam.
- Print at least one finding about how class performance changes across exams.


### 🛠️	Create and Save Trend Visualizations

#### Description
Use a single Python script to generate multiple charts that show individual student trends, whole-class trends, and score distributions across exams.

#### Requirements
Completed program should:

- Generate a line chart showing score trends for multiple students across exams.
- Generate a chart showing class-level trends such as average score by exam.
- Generate a distribution-style chart comparing score spread across exams.
- Save each chart as an image file.
- Use clear chart titles, axis labels, and legends where appropriate.
